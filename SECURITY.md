# Security Policy

## Supported Versions

This repository is a research project. No formal support guarantees are provided.

## Reporting a Vulnerability

If you believe you have found a security vulnerability related to this repository
(e.g., malicious code, compromised dependency instructions, or unsafe distribution artifacts),
please do **not** open a public issue.

Instead, contact the repository owner via GitHub profile messaging or email (if listed on the profile).
Please include:
- a clear description of the issue,
- steps to reproduce (if applicable),
- any relevant logs or screenshots,
- your recommended mitigation (if you have one).

We will acknowledge the report and take reasonable steps to address it.
